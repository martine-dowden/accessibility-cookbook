Alt Text

Overview & Demo

"1.4.9 Images of Text (No Exception): Images of text are only used for pure decoration or where a particular presentation of text is essential to the information being conveyed. (Level AAA)

Note: Logotypes (text that is part of a logo or brand name) are considered essential."

WCAG 2.0 Success Criteria 1.4.9

To allow users utilizing assistive technologies such as screen readers to know what images on a website represent, alternative text must be provided.

Demo

<img src="stairs.png">
<img src="img/stairs.png" alt="Pittock Mansion grand staircase, Oregon">

The HTML

The alt attribute should provide a description of the image. If content is found within the image, the content should also be available in the alt tag. Same for complex diagram. The same information that is conveyed in the diagram should be available to the user via the alt text.

<img src="stairs.png" alt="Dilapidated stairs.">

If the image does not add any additional information to the page, or is as a purely decorative image the alt text should be left blank.

<img src="border.png" alt="">

ARIA can instead be used to alert the screen reader of the image’s irrelevance to the content.

<img src="border.png" role="presentation">

Although role=”presentation” is not as widely supported as using a null alt attribute (W3C, 2015).

Another perk to proper use of alternative text on images is improved SEO.
