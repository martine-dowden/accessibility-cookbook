WCAG 2.0: Programmatic Content Structure Separation

Heading are important because they communicate the structure of the content. Furthermore, browsers, plug-ins, and assistive technologies use them for in-page navigation.

G141: Organizing a page using headings

Techniques for WCAG 2.0: G141
