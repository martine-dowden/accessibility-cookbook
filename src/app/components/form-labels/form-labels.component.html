Form Labels

WCAG

"Info and Relationships: Information, structure, and relationships conveyed through presentation can be programmatically determined or are available in text." (Level A)

WCAG 2.0 Success Criteria 1.3.1

"Labels or Instructions: Labels or instructions are provided when content requires user input." (Level A)

WCAG 2.0 Success Criteria 3.3.2

All users benefit from properly labeled fields. They help direct the user to what is expected of him / her. They further help assistive technology users because they are spoken by screen readers, and when properly placed in near proximity of the field they are associated with, assist users of screen magnifiers by increasing the likelihood of both the field and label being on the screen.

Elements such as radio buttons and checkboxes have the added benefit that properly associated labels increase the clickable area of the field.

The HTML

Using for and Id

"H44: Using label elements to associate text labels with form controls."

Techniques for WCAG 2.0: H44
Example

Code
<form>
  <label for="name">Name:</label>
  <input id="name" type="text" maxlength="64" name="name" value="">
  <br>
  <label ><input type="checkbox" name="human" value="">I am not a robot</label>
</form>

Using the title tag

H65: Using the title attribute to identify form controls when the label element cannot be used.

Techniques for WCAG 2.0: H65
Example
Phone number
Code
<form>
  <fieldset>
    <legend>Phone number</legend>
    <input id="areaCode" name="areaCode" title="Area Code" 
    type="text" size="3" value="" maxlength="3">
    <input id="exchange" name="exchange" title="First three digits of phone number" 
    type="text" size="3" value="" maxlength="3">
    <input id="lastDigits" name="lastDigits" title="Last four digits of phone number" 
    type="text" size="4" value="" maxlength="4">
    </fieldset>
</form>

Using fieldset and legend elements

H71: Providing a description for groups of form controls using fieldset and legend elements

Techniques for WCAG 2.0: H71
Example
The play Hamlet was written by:
Code
<form>
  <fieldset>
    <legend>The play <cite>Hamlet</cite> was written by:</legend>
    <input type="radio" id="shakesp" name="hamlet" checked="checked" value="a">
    <label for="shakesp">William Shakespeare</label>
    <br>
    <label>
      <input type="radio" name="hamlet" value="b">
      Rudyard Kipling
    </label>
  </fieldset>
</form>
